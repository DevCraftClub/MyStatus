Установка модуля MyStatus 1.0

Описание и возможности

Сей хак, не модуль, поможет вам отобразить статус любого сериала. Тем самым уведомляя автора и/или админа о статусе. Это поможет всё быстро и без проблем оформить новость и даст пользователям понять на какой стадии находится сейчас сериал, стоит ли его ждать или уже всё предрешено для него. Хак имеет несколько больших настроек, чем проставление статусов.

Скрипт ищет по АПИ фильм и проставляет сам ID к новости, однако поиск точен на 50%. Иногда встречаются сериалы, которые начинаются одинакого, типа: Бойтесь ходячих мертвецов или же ремейки, то будет всегда браться первое значение из списка.

О возможностях:

  • Модуль в версии UTF-8
  • Проверка данных по API MyShows
  • Проставление ID с MyShows
  • Проставление статуса сериала
  • Проставление даты выпуска последней серии, если сериал окончен
  • Проставление ID с TVRage, IMDB и КиноПоиска
  • Уведомление автора новости и/или админа о том, что статус сериала изменился
  • Если автор новости и админ одно лицо, то письмо отсылается один раз
  • Поднимает новость, если статус обновился

Статусы сериалов:

  • Снимается (returningseries) - если сериал уже выходит или ожидает нового сезона, то ставится этот сериал
  • Новинка (newseries) - если сериал впервые появился на MyShows, зачастую совпадает с новыми сериалами (опция отключается)
  • Пилотная серия (pilotordered) - если сериал стартует (или запускается следующий сезон), то часто выпускают пилотную серию, чтобы узнать мнения и она может отличаться от официальной первой серии (также отключается)
  • Закрыт (canceledended)- если сериал официально идёт на покой, то выводится этот статус
  • Приостановлен (onhiatus) - если сериал отправился по каким-то причинам, к примеру из-за смерти актёра или иного припятствия, на "паузу", то выставляется этот статус
  • Под вопросом (tbdonthebubble) - если у сериала спорные рейтинги или низкие показатели, что даже создатели не знают будет ли продолжение или нет, то этот статус будет выводится

Теги для подключения в файле fullstory.tpl

  • {myshows} - идентификатор на сайте myshows.me
  • {news-id} - стандартный тег дле, означающий идентификатор новости
  • {title} - вывод заголовка, стандартный тег
  • {statuss} - тег, который определяет стиль для статуса
  • {status} - вывод самого статуса

Теги в строке подключения

  • mysid - к нему подключается ID с MyShows, тег {myshows}
  • title - если в бд не указан ID, то скрипт будет искать по ID по АПИ и проставит в новость
  • news_id - обязательный тег, который подключает ID новости к скрипту
